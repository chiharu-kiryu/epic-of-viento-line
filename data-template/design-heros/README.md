# 英雄数据模板说明

`模板-英雄` 覆盖英雄源文档当前使用的核心属性、天生技能、四个普通技能、阳印、阴印、铸魔和铸神。

## 格式规则

- 第一行直接写英雄名，用作结构化文档标题。
- `别名` 和 `英文名` 分开填写；没有对应内容时保留空值。
- 所有字段统一使用中文全角冒号 `：`。
- 属性成长统一写成 `基础值 + 每级成长`，例如 `23 + 3.2`。
- 技能区块固定使用 `天生技能`、`技能1` 至 `技能4`。
- 每个技能至少包含 `名称`、`类型` 和 `描述`。
- 技能数值继续使用有语义的字段名，例如 `伤害`、`距离`、`持续时间`，不要统一改成无意义的“参数1”。
- 同一参数有多个等级时使用 `/` 分隔。
- `形态`、`分支`、`附加效果` 和 `子技能` 是可选结构，可按实际内容重复出现。
- 技能区块内部不得插入空行；空行只用于分隔顶层区块。当前解析器会把空行视为技能区块结束。
- 没有数值的字段可以删除，但顶层区块不得省略；确实不存在的技能或强化填写 `无`。

## 规范化映射

- `先天技能`、`天生技能:` → `天生技能：`
- `技能1:` 至 `技能4:` → `技能1：` 至 `技能4：`
- `cd`、`CD` → `冷却`
- `mp`、`MP` → `魔力消耗`
- 未标注的英雄第二行 → 按内容拆分到 `别名`、`英文名`
- 未标注的三行属性 → 依次补为 `力量`、`敏捷`、`智力`
