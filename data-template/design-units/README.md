# 单位数据模板说明

`模板-单位` 是 `design-units` 里单位文档的最大兼容模板。

## 格式规则

- 第一行是单位名。
- 优先保持字段顺序，未出现的字段留空即可。
- 全部字段使用中文全角冒号 `：`，必要时统一数值中的冒号写法。
- `被动`、`主动`、`技能` 为可重复段落块，可写多行。
- 对于 `击杀奖励`、`被动` 等有后续解释文本的字段，下一行若无冒号字段，可直接作为该字段的延续文本。
- 兼容 `攻击（远程）：` 或半角冒号 `:`，脚本会在标准化阶段归一化。

## 当前最大兼容覆盖

- 通用字段：`生命`、`攻击`、`护甲`、`魔抗`、`攻击间隔`、`移动速度`、`击杀奖励`
- 常见附加：`攻击类型`、`攻击距离`、`攻击范围`、`回血`、`状态抗性`
- 常见能力块：`被动`、`主动`、`技能`
