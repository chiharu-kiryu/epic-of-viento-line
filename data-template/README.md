# 数据格式模板

本目录存放各类源数据的最大兼容格式模板，用于统一 `design-data/` 中的原始文档。

模板不是业务文档，也不参与 `docs-standard/` 的结构化生成。每个子目录对应一个源数据类型，并尽量覆盖该类型现有文档的全部结构差异。

使用顺序：

1. 从同类源文档中统计共同字段和可选结构。
2. 在对应子目录中维护一份最大兼容模板。
3. 按模板统一原始文档的字段名、字段顺序和区块边界。
4. 运行标准化脚本生成结构化文档。
5. 验证结构化结果和页面渲染。
