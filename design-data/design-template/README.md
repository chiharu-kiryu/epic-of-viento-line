# 文档模板说明

当前模板目录提供两份基础模板：

- [模板-近战](/Users/Shared/chroot/dev/epic-of-viento-line/design-data/design-template/模板-近战)
- [模板-远程](/Users/Shared/chroot/dev/epic-of-viento-line/design-data/design-template/模板-远程)

## 使用建议

- 近战英雄优先从 `模板-近战` 复制。
- 远程英雄优先从 `模板-远程` 复制。
- 新模板若只是字段顺序不同，尽量在现有模板基础上扩展，避免出现过多分叉版本。

## 推荐补全方式

建议在模板基础上补齐以下字段标签：

- `力量：`
- `敏捷：`
- `智力：`
- `天生技能：`
- `技能1：`
- `技能2：`
- `技能3：`
- `技能4：`
- `阳印：`
- `阴印：`
- `铸魔：`
- `铸神：`

这样可以让英雄设计文档与 [design-data/README.md](/Users/Shared/chroot/dev/epic-of-viento-line/design-data/README.md) 中的统一规范保持一致。
